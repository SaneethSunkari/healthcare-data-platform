# Dashboard

Store Grafana provisioning, dashboards, and datasource definitions here.

Grafana is provisioned automatically from this folder when the Docker stack starts.
The dashboard URL is `http://localhost:3000/d/population-health-overview/population-health-overview`.
The patient drill-down dashboard URL is `http://localhost:3000/d/patient-360-overview/patient-360-overview`.
The risk directory dashboard URL is `http://localhost:3000/d/high-risk-patient-directory/high-risk-patient-directory`.
